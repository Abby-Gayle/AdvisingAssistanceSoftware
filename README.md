# AdvisingAssistanceSoftware
The problem I was solving in this software was creating an academic advising help tool. The program should load the data from a file, print the courses in order, and search a specific course data. I broke what needed to be done into smaller steps and then I had decided I would use a hash tables to make memory better.
Some roadblocks I came across was I ran into infinate loops when inputting data. Having the file be able to be read correctly, and sorting and printing the data. The way I had approached the problems was first for the infinate loop the problem was the blank spaces. So I had to use a getline to get the blank space if any when adding file name. After that I added a clear to flush out the data because when I pressed enter something would else would heappen this helped.
Working on this project helpped me understand how to think of memory and how to use hash tables this was a new concept for me. This project made me realize just how important it is to decompress functions. The way I did my code I felt like I made some methods do too much makeing it a harder to read and reuse. Not that I can't be reused but creating better looking software would go a long way.
